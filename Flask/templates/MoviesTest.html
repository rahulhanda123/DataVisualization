And here it is! Movies data visualized with D3,DC, crossfilter and python libraries. The data was collected by scraping iMDB, oMDB and Rotten Tomatoes. It was minimally cleaned up by filtering for movies that have rating values missing and are in English. The data is presented Genre wise. As you can see this chart visualizes the number of genre movie released between 2000 and 2015, as well as the total number of movies in those years. A single movie may be associated with zero, one, or multiple genres and so the ‘Total Movies’ line corresponds to actual movie counts in that genre and You can select which genre you want to view in the drop down option at the top of the page. The rest of the date then adjusts for that particular genre, The clear conclusion is that hollywood follows a fixed formula as far as movie genres are concerned. The highest earning movies on average belong to Action,Adventure,Fantasy and Sci-fi Genres. Bit of Escapists arent we?

There has been an explosion in film production from 2004 onwards. If we select the timeline 2004-2012 from the below line chart, surprisingly the largest benificiary from this is the Drama genre. The point of making such an open-ended visualization is to encourage the viewer to find such patterns by themselves. There is a ton more that can be done with the data. The direction I decided to go with it was to show some example trends and then allow the user to discover more about the data. I would love to eventually add controls to view any year range for all the following charts, but they still reveal some interesting aspects about modern movies and IMDB metrics. For imdb ratings we see a nice bell curve. This is expected as most movies are rated average by majority of the populations and there are rare movies with high ratings.
We can also interact with the tomato consensus pie chart and see how the bell curve changes with respect to fresh ,consensus and rotten ratings. Again we see a clear correlation between the voters on the two sites as rotten movies tend to have lower ratings on imdb as well.
